# Windows Update.ps1

Egress Opening: Opens outbound TCP 80/443.

Search: Queries for all uninstalled software updates.

Filter: Sorts the list to include only those categorized as "Security" or "Critical."

Download: Pulls the update packages from the update source.

Install: Applies the patches to the system.

Verify: Reports whether the installation was successful and if a manual reboot is pending.
